# FLINK

The FLINK project is a collection of some useful fortran modules and subroutines, which are widely used in the other numerical programs. 

## Version

v0.9.0 (devel)

## License

GNU General Public License Version 3

## Features

* Modules
* Subroutines

## Installation

```sh
$ cd flink/build
$ editor make.sys
$ make
```
